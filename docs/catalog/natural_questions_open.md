# `natural_questions_open`

*   **Description**:

The NQ-Open task, introduced by Lee et.al. 2019, is an open domain question answering benchmark that is derived from Natural Questions. The goal is to predict an English answer string for an input English question. All questions can be answered using the contents of English Wikipedia.

*   **Homepage**: [https://github.com/google-research-datasets/natural-questions/tree/master/nq_open](https://github.com/google-research-datasets/natural-questions/tree/master/nq_open)

*   **Source code**: [`tfds.question_answering.NaturalQuestionsOpen`](https://github.com/tensorflow/datasets/tree/master/tensorflow_datasets/question_answering/natural_questions_open.py)
